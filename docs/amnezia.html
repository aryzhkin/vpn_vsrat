VPN через AmneziaVPN

Самый простой способ — 2 шага на любом устройстве

Работает на телефоне и компьютере. Весь трафик идёт через VPN — YouTube, Instagram, Discord, всё.

AmneziaVPN — бесплатное приложение с открытым кодом. Использует протокол AmneziaWG — модифицированный WireGuard, который обходит блокировки в России.

Шаг 2. Импортируй конфиг

2
Скопируй ключ и вставь в приложение

Я пришлю тебе строку вида vpn://...

Скопируй её целиком, открой AmneziaVPN и нажми кнопку с + или Добавить подключение. Приложение само распознает конфиг из буфера обмена.

Если не распознало — нажми Добавить через файл или текст и вставь строку вручную.

На iPhone также можно открыть .conf файл через «Поделиться» → AmneziaVPN.

Готово. Как проверить?

Нажми Подключиться в приложении. Затем открой браузер и загугли «какой у меня IP» — должен показать 194.154.26.53, а не твой домашний адрес.

Не работает?

Если AmneziaWG не подключается — напиши мне, дам ключи для другого протокола (VLESS / Hysteria2).

AmneziaWG работает через UDP. Если твой провайдер блокирует UDP целиком (бывает на Tele2), поможет VLESS — он работает через TCP/443 и маскируется под обычный HTTPS.

Работает через AmneziaVPN — протокол AmneziaWG (обфусцированный WireGuard)
