VPN через Clash Verge Rev

Умный VPN на компьютере — выбираешь что идёт через VPN, а что напрямую

Через VPN пойдут только заблокированные сервисы (Telegram, Discord, YouTube, Instagram и т.д.).
Всё остальное (игры, банки, обновления) — напрямую, без потери скорости и пинга.

Clash Verge Rev — бесплатное приложение с открытым кодом. Работает на Windows, macOS и Linux. Использует протокол VLESS Reality — маскируется под обычный HTTPS-трафик, блокировки в России не видят.

Шаг 1. Скачай Clash Verge Rev

1
Скачай установщик для своей системы

Скачать с GitHub

Windows: скачай файл Clash.Verge_x64-setup.exe и установи.
macOS: скачай .dmg файл (arm64 для Mac 2020+, x64 для старых).

Windows может показать синее окно SmartScreen — нажми Подробнее → Выполнить в любом случае.

Шаг 2. Установи службу (Service Mode)

2
Это нужно для перехвата трафика приложений

Открой Clash Verge → Настройки (слева) → найди Служба (Service Mode) → нажми Установить.
Подтверди права администратора (UAC). Это одноразовая настройка.

Без службы можно использовать Системный прокси — будет работать для браузера, Telegram и Discord. Но TUN-режим (полный перехват трафика) требует службу.

Шаг 3. Импортируй конфиг

3
Я пришлю тебе файл (.yaml)

Открой Clash Verge → Профили (слева) → Создать → Local.
Вставь содержимое файла, который я тебе прислал, и нажми Сохранить.
Нажми на созданный профиль, чтобы он стал активным (подсветится синим).

Шаг 4. Включи VPN

4
Выбери режим работы

На главной странице:

  • Режим работы: Rule (автоматически выбирает что через VPN, а что напрямую)
  • Включи Системный прокси или Режим TUN (если установлена служба)
Системный прокси — проще, работает для большинства приложений.
Режим TUN — перехватывает вообще весь трафик, надёжнее, но требует службу.

Что идёт через VPN?

В конфиге уже настроены правила. Через VPN автоматически идут:

  • Telegram
  • Discord
  • YouTube
  • Instagram
  • Facebook
  • Twitter / X

Всё остальное идёт напрямую — игры, банки, Яндекс, обновления Windows и т.д.

Нужно добавить ещё сайт через VPN? Напиши мне, обновлю конфиг.

Готово. Как проверить?

Открой браузер → загугли «какой у меня IP». Должен показать 194.154.26.53 (наш сервер в Нидерландах).

Попробуй открыть YouTube или Instagram — должны работать. Обычные сайты (Яндекс, банки) должны открываться как раньше, без задержек.

Не работает?

Проверь:

  • Профиль активирован (подсвечен синим в разделе Профили)?
  • Включён Системный прокси или Режим TUN?
  • Режим работы стоит Rule?

Если всё правильно, но не работает — напиши мне.
