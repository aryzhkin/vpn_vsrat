VPN только в браузере

Для геймеров и тех, кто использует Zapret / Booster

Zapret и Booster продолжат работать как обычно — конфликтов нет.
Игры идут напрямую (пинг не меняется), Discord через Zapret как раньше, браузер — через VPN.

Нужно установить программу-прокси (работает в фоне) и настроить расширение для браузера (переключает трафик).

Шаг 1. Установи прокси

Выбери свою систему:

1
Скачай v2rayN

Открыть страницу загрузки →
Скачай файл v2rayN-windows-64.zip, распакуй в любую папку и запусти v2rayN.exe

2
Импортируй ключ

Скопируй ссылку vless://... которую прислал тебе я.
В v2rayN нажми Servers → Import bulk URL from clipboard

3
Выбери режим прокси

Внизу окна v2rayN переключи режим на Nothing — это важно, чтобы VPN работал только в браузере, а не на всём компьютере.

4
Подключись

Выбери сервер в списке и нажми Enter. Внизу окна появится статус подключения.
v2rayN создаст локальный прокси на порту 10808 — его мы укажем в расширении браузера.

1
Скачай V2RayXS

Открыть страницу загрузки →
Скачай .dmg файл, открой и перетащи в Applications

2
Импортируй ключ

Скопируй ссылку vless://... которую прислал тебе я.
Нажми на иконку V2RayXS в меню → Import from pasteboard

3
Выбери режим Manual

Нажми на иконку в меню → Manual Mode, затем Turn v2ray-core On.
В Manual Mode прокси работает на порту 1080 — его мы укажем в расширении браузера.

Права администратора не нужны. Никакие системные настройки не меняются — это просто локальный прокси для браузера.

Шаг 2. Установи расширение для браузера

Выбери свой браузер:

1
Установи расширение ZeroOmega

Открыть в Chrome Web Store →

2
Открой настройки расширения

Нажми на иконку расширения в браузере (вверху справа) → Options

3
Открой профиль proxy

Слева в списке уже есть профиль proxy — нажми на него

4
Введи настройки

В поле Protocol выбери SOCKS5
Server: 127.0.0.1
Port: 10808 (Windows) или 1080 (macOS)
Bypass List трогать не нужно — оставь как есть

5
Сохрани

Нажми зелёную кнопку Apply changes слева внизу

6
Включи VPN в браузере

Нажми на иконку расширения — появится меню. Выбери proxy (не путай с [System Proxy]).
Чтобы выключить — выбери [Direct].

Готово. Как проверить что работает?

Открой в браузере поиск и введи «какой у меня IP» — должен показать 209.38.42.79, а не твой домашний адрес.

Прокси работает через XRay — протокол VLESS с маскировкой под обычный HTTPS
