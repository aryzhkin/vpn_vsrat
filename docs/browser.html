VPN только в браузере

Остальные программы работают без VPN

Как это работает

Outline создаёт локальный SOCKS5 прокси на твоём компьютере (обычно 127.0.0.1:1080). Можно настроить браузер использовать именно его — тогда только браузер идёт через VPN, остальное — нет.

Discord, Telegram и другие приложения при этом работают напрямую (без VPN). Если нужен Discord через VPN — включай Outline как обычно.

Шаг 1. Установи и подключи Outline

Сначала нужен работающий Outline на компьютере:

Скачать для Windows Скачать для macOS
  1. Установи Outline
  2. Добавь ключ ss://... и подключись
  3. Outline работает в фоне и держит прокси на 127.0.0.1:1080
Outline должен быть запущен и подключён — иначе прокси не работает.

Шаг 2. Настрой браузер — расширение SwitchyOmega

Это самый удобный способ — можно быстро переключаться.

  1. Установи расширение Proxy SwitchyOmega:
    Chrome Web Store →
  2. Открой настройки расширения → New Profile → тип Proxy Profile
  3. Заполни:
    • Protocol: SOCKS5
    • Server: 127.0.0.1
    • Port: 1080
  4. Нажми Apply changes
  5. В иконке расширения выбери созданный профиль — всё, браузер идёт через VPN
Чтобы отключить — выбери в расширении профиль Direct.
  1. Установи расширение Proxy SwitchyOmega:
    Firefox Add-ons →
  2. Открой настройки расширения → New Profile → тип Proxy Profile
  3. Заполни:
    • Protocol: SOCKS5
    • Server: 127.0.0.1
    • Port: 1080
  4. Нажми Apply changes
  5. В иконке расширения выбери созданный профиль
Чтобы отключить — выбери профиль Direct.

Альтернатива: встроенные настройки системы (без расширения)

Можно прописать прокси на уровне Windows/macOS — тогда большинство программ пойдёт через VPN (но не все).

  1. Настройки → Сеть и Интернет → Прокси
  2. Ручная настройка прокси → включить
  3. Адрес: 127.0.0.1, порт: 1080
  4. Сохранить
  1. Системные настройки → Сеть → выбери своё соединение → Подробнее → Прокси
  2. Включи SOCKS-прокси
  3. Сервер: 127.0.0.1, порт: 1080
  4. ОК → Применить
Этот способ не даёт быстро переключаться. Расширение SwitchyOmega удобнее.
