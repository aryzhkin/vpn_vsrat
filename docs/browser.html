VPN только в браузере

Для геймеров и тех, кто использует Zapret / Booster

Zapret и Booster продолжат работать как обычно — конфликтов нет.
Игры идут напрямую (пинг не меняется), Discord через Zapret как раньше, браузер — через VPN.

Для этого нужно установить два приложения: Outline (работает в фоне) и расширение для браузера (переключает трафик).

Шаг 1. Установи Outline

Скачать для Windows Скачать для macOS
  1. Установи и открой Outline
  2. Нажми «Добавить сервер» и вставь ключ ss://..., который прислал тебе я
  3. Нажми «Подключиться»
  4. Outline можно свернуть — пусть работает в фоне
Outline должен быть запущен и подключён, пока ты хочешь использовать VPN в браузере.

Шаг 2. Установи расширение для браузера

Выбери свой браузер:

1
Установи расширение ZeroOmega

Открыть в Chrome Web Store →

2
Открой настройки расширения

Нажми на иконку расширения в браузере (вверху справа) → Options

3
Создай новый профиль

Слева нажми New profile... → придумай название (например «VPN») → тип оставь Proxy Profile → нажми Create

4
Введи настройки

В поле Protocol выбери SOCKS5
Server: 127.0.0.1
Port: 1080

5
Сохрани

Нажми Apply changes слева внизу

6
Включи VPN в браузере

Нажми на иконку расширения → выбери профиль «VPN».
Чтобы выключить — выбери Direct.
