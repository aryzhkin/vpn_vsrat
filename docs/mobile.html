VPN на телефоне

Шаг 1. Скачай приложение

Выбери свою систему:

Скачай v2rayNG — это бесплатное приложение для VPN-подключения:

Google Play

Нет Google Play? Скачать APK с GitHub

Скачай Hiddify — бесплатное приложение для VPN (поддерживает VLESS и Hysteria2):

App Store

Шаг 2. Получи ключ

Напиши мне — пришлю личный ключ. Он выглядит как длинная ссылка, начинающаяся с vless://

У каждого свой ключ — не передавай его другим.

Шаг 3. Подключись

1
Открой v2rayNG

Нажми + вверху справа → Import config from Clipboard (Импорт из буфера обмена)

2
Вставь ключ

Скопируй ссылку vless://... — приложение подхватит её автоматически

3
Подключись

Нажми большую кнопку V внизу экрана. При первом запуске разреши создание VPN-подключения.

1
Открой Hiddify

На главном экране нажми + (новый профиль) → Add from clipboard

2
Вставь ключ

Скопируй ссылку vless://... — сервер появится в списке

3
Подключись

Нажми большую кнопку подключения в центре экрана. При первом запуске разреши добавить VPN-конфигурацию.

Когда VPN включён, вверху телефона появляется иконка VPN. Чтобы выключить — открой приложение и нажми кнопку отключения.

Не работает? Попробуй запасной протокол

Если основной ключ vless:// не подключается — напиши мне, пришлю запасной ключ hy2:// (Hysteria2). Он работает по-другому (UDP вместо TCP) и обходит блокировки, которые ловят основной протокол.

Импортируется точно так же — скопируй ссылку hy2://... и вставь в v2rayNG / Hiddify тем же способом (из буфера обмена).

Как проверить что всё работает?

Открой браузер на телефоне и загугли «какой у меня IP».
Должен показать 194.154.26.53 — это наш сервер в Нидерландах.

Работает через XRay — протокол VLESS с маскировкой под обычный HTTPS
