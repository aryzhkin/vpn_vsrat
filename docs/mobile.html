VPN на телефоне

1. Скачай Outline

Установи официальное приложение Outline:

App Store (iOS) Google Play (Android)

Нет Google Play? Скачать APK напрямую

2. Получи ключ

Ключ выглядит так: ss://XXXXXX@209.38.42.79:20454/...

Попроси его у меня — пришлю в личку. Один ключ = один человек.

3. Добавь ключ в Outline

  1. Открой приложение Outline
  2. Нажми «Добавить сервер» (или кнопку +)
  3. Вставь ключ в поле — он определится автоматически
  4. Нажми «Подключиться»
На iOS при первом подключении появится запрос на добавление VPN-профиля — нажми «Разрешить».

4. Проверь

  1. Загугли «какой у меня IP»
  2. Должен показать IP сервера (209.38.42.79), а не твой домашний
Zapret / Booster: отключи их когда VPN включён — они могут мешать маршрутизации. С VPN Discord и другие заблокированные сервисы работают и так.

Включать VPN не для всего трафика?

На мобилке Outline работает как системный VPN — весь трафик идёт через него. Выборочно по приложениям нативно не поддерживается.

Если хочешь только браузер — смотри инструкцию для десктопа (на телефоне это не настраивается удобно).
