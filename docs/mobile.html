Шаг 2. Получи ключи

Напиши мне — пришлю два ключа:

  • vless://... — основной протокол (TCP, маскируется под обычный HTTPS)
  • hy2://... — запасной протокол Hysteria2 (UDP, другой способ обхода блокировок)

Это два разных протокола на разный случай. Импортируй оба — если один перестанет работать, переключись на другой.

У каждого свой ключ — не передавай его другим.

Шаг 3. Подключись

1
Открой Hiddify

На главном экране нажми + (новый профиль) → Add from clipboard

2
Импортируй оба ключа

Скопируй ссылку vless://... и добавь. Потом скопируй hy2://... и добавь тем же способом. В списке появятся два сервера.

3
Подключись

Выбери один из серверов и нажми большую кнопку подключения. При первом запуске разреши добавить VPN-конфигурацию.

Подключись

Выбери один из серверов и нажми большую кнопку подключения. При первом запуске разреши добавить VPN-конфигурацию.

Когда VPN включён, вверху телефона появляется иконка VPN. Чтобы выключить — открой приложение и нажми кнопку отключения.

Не работает?

Если один сервер не подключается — попробуй другой. У тебя два ключа на разных протоколах:

  • vless:// — TCP, маскируется под HTTPS. Стабильнее в обычных условиях.
  • hy2:// — UDP, быстрее обходит блокировки, если TCP режут.

Если оба не работают — напиши мне.

Как проверить что всё работает?

Открой браузер на телефоне и загугли «какой у меня IP».
Должен показать 194.154.26.53 — это наш сервер в Нидерландах.

Работает через XRay (VLESS) и Hysteria2 (UDP)
