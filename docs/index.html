Подключение к VPN

Свободный интернет для своих

Телефон (iOS / Android) Только браузер (Windows / Mac)

Шаг 1. Скачай приложение Outline

Выбери своё устройство:

iOS (iPhone) Android macOS Windows

Шаг 2. Добавь ключ

  1. Получи ключ доступа от меня — это ссылка, начинающаяся с ss://
  2. Открой Outline — он должен подхватить ключ автоматически
  3. Если не подхватил — нажми + и вставь ключ вручную

Шаг 3. Подключись

  1. Нажми «Подключиться»
  2. Проверь: загугли «какой у меня IP» — должен показать IP сервера, а не твой домашний
Если используешь Zapret или Booster — отключи их когда VPN включён, иначе могут быть конфликты.

Хочешь VPN только в браузере?

Если не хочешь пускать через VPN весь трафик — можно настроить только браузер. Это чуть сложнее, но работает.
